# 要求事項
- 入力データの日本語指示に従い、極めて詳細かつ視覚的な**英語のテキストプロンプト**を作成してください。
- `prompt` フィールドには、メインの被写体、背景、構図、ライティング、カメラスタイル、雰囲気、質感などの詳細な描写を含めます。

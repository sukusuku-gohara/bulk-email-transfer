# タスク
提供された3つのアイキャッチ画像アイデア（JSON）を元に、ラフ画像（プレビュー）を生成してください。
この段階では「構図やモチーフが視覚的に確認できるラフイメージ」を作ることが目的です。
トークン消費を削減する形で最低限構図やメイン見出し（画像内文字）、配置がわかるようにしてください。

#ラフで見るポイントは以下
- 文字の大きさ感
- モチーフの位置
- 背景の質感

# 入力データ（アイデアJSON）
```JSON
{{IDEATION_JSON}}
```

# 利用するモデル
- Nano Banana (Gemini 2.5 Flash Image)

# 要件
- 出力は各アイデアに対する `aspect_ratio`, `prompt` (日本語), `negative_prompt` (日本語) のリストです。
- `prompt` には、被写体、背景、ライティング、カメラスタイル、雰囲気（ムード）を詳細に日本語の文章で記述してください。
- 各アイデアの `do` および `visual`, `motif`, `composition`, `mood` の内容をしっかりと反映させてください。
- `negative_prompt` には、アイデアの `dont` 指定事項や、一般的な低品質画像を避けるためのキーワード（worst quality, blurry, mutatedなど）を含めてください。
- 全体的な禁止事項: {{BANS}}

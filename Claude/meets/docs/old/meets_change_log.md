### CHG-20260309-001: 会員絞り込み機能の拡充

| 項目 | 内容 |
|------|------|
| **変更ID** | CHG-20260309-001 |
| **変更日** | 2026-03-09 |
| **起票者** | ユーザー指示 |
| **変更種別** | 仕様変更 |
| **カテゴリ** | 業務 |
| **影響度** | 中 |
| **ステータス** | 反映中 |

#### 変更前の仕様

- エージェントの会員検索条件は年齢・年収・地域・学歴・結婚歴等に限定（BR-010〜BR-013で定義）
- BR-006記載のプロフィール項目の一部のみが検索フィルタに利用可能
- 交際ステータスは検索条件に含まれない

#### 変更後の仕様

- **個人の登録情報（BR-006記載の全項目）を全て絞り込み対象とする**
  - 本人情報: ID/写真/ニックネーム/氏名/性別/年齢/居住地/出身地/国籍/身長/体型/血液型/兄弟姉妹/趣味/酒タバコ/職業/年収/休日/学歴/資格/結婚歴/子供有無/家族構成/恋愛・結婚について/自己PR文/担当エージェントからの紹介文
- **交際ステータスも絞り込み対象に追加する**
  - お見合い待ち/仮交際中/本交際中等のステータスで絞り込み可能

#### 変更理由

エージェントがより効率的に会員のマッチング候補を探せるよう、絞り込み条件を大幅に拡充する。全プロフィール項目を検索対象にすることで、より精度の高いマッチング提案が可能になる。

### CHG-20260309-002: 証明書類の有効期限設定

| 項目 | 内容 |
|------|------|
| **変更ID** | CHG-20260309-002 |
| **変更日** | 2026-03-09 |
| **起票者** | ユーザー指示 |
| **変更種別** | 仕様追加 |
| **カテゴリ** | 業務 / コンプライアンス |
| **影響度** | 中 |
| **ステータス** | 反映中 |

#### 変更前の仕様

- BR-003（追加証明書アップロード）: 独身証明/年収証明/学歴証明のアップロード機能。発行日・有効期限に関する条件は未定義
- BR-004（年収証明の条件付き必須化）: 年収600万円以上の会員に年収証明書の提出を必須化。有効期限に関する条件は未定義

#### 変更後の仕様

- **独身証明書・住民票**: 発行日から3ヶ月以内のものに限る。3ヶ月を超過した証明書はアップロード時にエラーとする
- **年収証明書**: 発行日から1年以内のものに限る。1年を超過した証明書はアップロード時にエラーとする
- 既にアップロード済みの証明書が有効期限切れとなった場合、「証明書の有効期限が切れました。再提出してください」通知を送信し、バッジを一時的に非表示にする

#### 変更理由

証明書類の信頼性を確保するため、有効期限を設定する。独身証明書・住民票は短期間での変動可能性があるため3ヶ月、年収証明書は年次発行が一般的であるため1年とする。

### CHG-20260309-003: 決済モデルの変更

| 項目 | 内容 |
|------|------|
| **変更ID** | CHG-20260309-003 |
| **変更日** | 2026-03-09 |
| **起票者** | ユーザー指示 |
| **変更種別** | 仕様変更 |
| **カテゴリ** | 業務 / 決済 |
| **影響度** | 高 |
| **ステータス** | 反映中 |

#### 変更前の仕様

- **C-3**: 会員の月額会費は980円固定。エージェント所属後もmeetsが徴収
- **BR-030**: 全会員一律980円/月。初回お見合い申し込み時に課金開始。エージェント所属後もmeetsが徴収
- **BR-031**: 毎月1日に全アクティブ会員に自動請求
- **D-5**: 月額会費型の決済モデル。エージェント不在時：meets側が手数料徴収。エージェント紹介時：エージェント側が売上で、meets側が10%を手数料として徴収
- **MRR計算式**: 会員月額980円 x アクティブ会員数 + エージェント売上の10%

#### 変更後の仕様

- **仲人（エージェント）未登録ユーザー**: 会員本人から一律980円/月を徴収（サブスク）。課金トリガーは従来通り初回お見合い申し込み時
- **仲人付き会員**: 会員本人への直接課金はなし。代わに**仲人（エージェント）からシステム利用料として担当会員1人あたり980円/月を徴収**する
- 980円という金額は変更可能性あり [ASSUMPTION: 当面は980円で運用]
- エージェントへのシステム利用料請求は、担当会員数に基づき毎月自動で集計・請求

#### 変更理由

仲人（エージェント）が担当する会員に対しては、会員本人ではなくエージェント側にシステム利用料を負担させることで、エージェントのビジネスモデルとしての自律性を高め、会員のエージェント所属インセンティブを強化する。会員にとっては「エージェント所属 = 月額無料」となり、エージェント所属率の向上が期待できる。
